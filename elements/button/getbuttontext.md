# getText

## **Syntax:**

```lua
text getText(button)
```

#### _**\#Counterpart:**_ [_**setText**_](setbuttontext.md)

### **Parameters:**

* **button** \(element\) : Button element you wish to retrieve the text of.

### **Returns:**

* **text** \(string\) : Text of the button.

## **Example:**

```lua
local buttonText = beautify.button.getText(createdButton)
print(buttonText)
```
